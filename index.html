Welcome to MRTL Auto-Trading

MRTL is a fully automated trading bot designed to simplify and streamline trading for retail traders. Built around technical strategies, custom indicators, and algorithmic execution, the MRTL Bot helps users identify entries and exits automatically and even execute trades through broker connections. The bot is designed to run 24/7 and can be used with platforms like TradingView, Tradovate, and Rithmic. It sends trade signals through alerts, which are then automatically executed using a webhook + script setup hosted online (like via Replit). Our goal is to provide a plug-and-play solution for those who want to automate profitable trading strategies without needing to code or manage complex infrastructure.

How It Works

You receive a ZIP file or hosted Replit link that connects TradingView alerts to your broker (like Tradovate or Rithmic). You plug in your API keys and the bot trades for you.

  • Set up takes less than 5 minutes.
  • Customize the strategy within TradingView Pine Script.
  • Use your own webhook or ours.
